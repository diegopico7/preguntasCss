    10- Propiedad display en css
    block = ocupan todo el espacio a lo ancho del contenedor y generan salto de línea
    inline = ocupan el espacio segun su contenido, no generan saltos de línea, no admiten dimensiones width y height
    inline-block = un intermedio entre block e inline no ocupa todo el ancho disponible y se le puede dar dimensiones width y height
    none = oculta el elemento, por lo que no ocupara ningun espacio. solo oculta no se borra el elemento
    flex = cajas flexibles ideal para maquetar componentes, es unidemensional se trabaja solo con 1 columna o 1 fila
    Grid = para maquetar layout, es bidimensional podemos trabajar con varias columnas o filas a la vez
    6- Tipos de posicionamiento en css
    • Static = posicionamiento por defecto, y se posiciona en el flujo normal del documento
    • Relative = ocupa su espacio en el documento, el elemento es posicionado relativo a su posicion normal, y con las propiedades top , right, bottom, left podemos moverlo, al moverlo el elemento conseva su espacio reservado
    • Absolule = se posiciona respecto al ancestro mas cercano que tambien este posicionado, si no hay ninguno toma al viewport como referencia, pierde su espacio en el flujo del documento y los elementos posicionados absolute pueden moverse y superponerse sobre otros elementos
    • Fixed= el elemento se posiciona en forma fija y respecto al viewport, y ojo que puede tapar otros elementos, al hacer scroll este seguira fijo en la pantalla, ocupa solo el espacio de su contenido a menos que le demos nosotros valores a width y height
    • Sticky= puede considerarse un intermedio entre relativo y fixed, se utiliza cuando queremos que un elemento se posicione en un lugar especifico, y que cuando lleguemos a ese elemento se quede en forma fija, las propiedades top, right, bottom, left no se usan para mover el elemento, si no que se usan para indicar en que punto el elemento tendra un compratamiento diferente, antes de llegar a ese punto el elemento va a comportarse de manera natural
